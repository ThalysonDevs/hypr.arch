# Hypr.arch 🐧

Setup automático completo para Arch Linux + Hyprland, pronto para uso e desenvolvimento.

### Como instalar so cola no terminal!
```bash
git clone https://github.com/ThalysonDevs/hypr.arch.git
cd hypr.arch
chmod +x install.sh
./install.sh
````
